Τι να περιμένετε

Ξεκινάμε από αυτό που είναι πιο «ζωντανό» για εσάς τώρα. Βάζουμε στόχους που βγαίνουν στην καθημερινότητα, χωρίς να πιέζουμε την αλλαγή.

Σκοπός δεν είναι να γίνετε «τέλειοι». Είναι να νιώθετε πιο σταθεροί και πιο ελεύθεροι μέσα σε αυτό που ζείτε.

Σχετικά

Μερικές φορές δεν είναι ένα «μεγάλο» γεγονός που μας φέρνει εδώ. Είναι η συσσώρευση. Η κούραση που μένει, το σώμα που σφίγγει, οι σκέψεις που γυρίζουν την ίδια διαδρομή.

Η θεραπεία, για μένα, είναι ένας ασφαλής τόπος. Ένας χώρος όπου μπορείτε να ακουστείτε με σεβασμό, χωρίς να χρειάζεται να αποδείξετε τίποτα. Δουλεύουμε με αυτό που είναι αληθινό για εσάς, και το κάνουμε με τρόπο που να μπορεί να «ζήσει» μέσα στην καθημερινότητα.

Για τους γονείς, το ζητούμενο δεν είναι η τελειότητα. Είναι η στήριξη, η κατανόηση, και μια σταθερή συνεργασία ώστε το σπίτι να γίνει πιο ήρεμο, πιο προβλέψιμο, πιο ανθρώπινο.

Ο χώρος

Ένας ήρεμος χώρος στη Ρόδο, σχεδιασμένος για να νιώθετε ασφαλείς και άνετα.

Μικρές λεπτομέρειες, ζεστό φως και σταθερό πλαίσιο, για να μπορεί να υπάρξει χώρος για σκέψη και συναίσθημα.
